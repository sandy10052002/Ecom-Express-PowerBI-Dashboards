
Ecom Express Power BI Dashboard

Project Overview

This project is an interactive Power BI dashboard developed for analyzing Ecom Express operational and shipment data. The dashboard provides business insights through data visualization and KPI tracking.

Tools Used

- Power BI
- SQL
- Excel

Dashboard Insights

- Total Shipments Analysis
- Delivery Performance Tracking
- Monthly Shipment Trends
- Region-wise Analysis
- Order Status Monitoring
- KPI Visualization

Features

- Interactive Dashboard
- Dynamic Filters and Slicers
- Visual Reports and Charts
- Business Performance Insights

Files Included

- Power BI Dashboard (.pbix)
- Dataset File
- Dashboard Screenshot
